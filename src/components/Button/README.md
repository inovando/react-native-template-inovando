# <Button />

## Usage

```javascript
function MyCoolComponent() {
  return (
    <Button
      // some form is submitting or screen is loading?
      loading

      // do not enable this button "onPress" function
      disabled

      // like "fullWidth", will fulfill button into layout
      stretch

      // outline style for secondary actions
      outline
    >
      Some Text
    </Button>
  )
}
```
